Tool Bar displays the Runtime Meta-Data JSON and Data JSON retrieved from a data source and assigned to a data-bound property:

  • Assign a data source to the Tool Bar.
  • In the Properties view, enter a selection string (or use the Select Data dialog to create one) in the input field of one of the data-bound properties ResultCell, ResultCellList, ResultCellSet, or ResultSet.
  • Pick the data-bound property whose JSONs the Tool Bar shows with the Show Data-Bound Property property.
  • Format the JSONs nicely with the Pretty Print property.
